# Milikan Oil Drop Lab

Eksperimen ini mengikuti tahapan ilmiah FisikaSeru dari preliminary hingga laporan akhir.

## Struktur
- Stage 0: orientasi, asumsi, gate.
- Stage 1: recall konsep.
- Stage 2: pengambilan data (IndexedDB).
- Stage 3: analisis & plotting.
- Stage 4: laporan & ekspor PDF (login).
- Post-test: refleksi.

## Catatan
Seluruh formula fisika berada di `core/` dan diekspor melalui `public/js/core/physics-core.js`.
